# 评价Vue开发

## 产品介绍
###
七彩评价是一个面向小学生的行为习惯养成管理平台，通过学校、老师、家庭、孩子多方评价来监督孩子行为习惯培养。
###

## 项目设置
```
npm install
```

### 编译并重新加载以进行开发
```
npm run serve
```

### 打包代码，为生产而编译和缩小
```
npm run build
```

### 运行eslint格式化代码
```
npm run lint
```
